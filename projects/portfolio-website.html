Responsive Portfolio Website

Implementation notes

  • Static architecture: HTML/CSS/JS on GitHub Pages — low latency, no frameworks.
  • Performance:
    • Lazy images + explicit dimensions prevent CLS.
    • Single deferred JS; starfield capped, debounced, paused when hidden.
    • CSS transitions (no heavy libraries) keep overhead under 2 ms.
  • Accessibility:
    • Semantic landmarks & descriptive link text.
    • Alt attributes for all imagery.
    • ESC & click-away lightbox dismissal.
  • Data & build:
    • Repo snapshot documents deployments & language split.
    • Planned: inline critical CSS + small service worker + Lighthouse logging.
  • Structure: lowercase filenames; body class themes.
