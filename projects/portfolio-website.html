Responsive Portfolio Website

Implementation notes

  • Static architecture: HTML/CSS/JS on GitHub Pages — low latency, no frameworks.
  • Performance:
    • Lazy images + explicit dimensions to prevent CLS and reduce initial payload.
    • Single deferred JS; starfield capped, debounced resize, paused when hidden.
    • CSS class-based transitions replace heavier libraries (under 2 ms overhead).
  • Accessibility:
    • Semantic landmarks, descriptive link text, comprehensive alt attributes.
    • Themed palettes tuned for contrast (target WCAG AA).
    • Lightbox dismissable via ESC and click-away; no focus traps.
  • Data & build:
    • Repository snapshot documents deployments & language split.
    • Planned: inline critical CSS, small service worker, Lighthouse metric logging.
  • Structure: lowercase filenames; assets in images/; themes via body classes.
