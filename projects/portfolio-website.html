Responsive Portfolio Website

Implementation notes

  • Static architecture: HTML/CSS/JS on GitHub Pages — minimal latency.
  • Performance:
    • Lazy images + fixed dimensions prevent CLS.
    • Single deferred JS; animation paused when tab hidden.
    • Lightweight CSS transitions instead of libraries.
  • Accessibility: semantic landmarks, clear alt text, ESC/click-away lightbox.
  • Data & build: repository snapshot planned (deployments & language split); future inline critical CSS + SW.
  • Structure: lowercase filenames; theme via body classes.
