Flight Path Simulation (Python)

Simulation details

The simulation models a simplified point-mass aircraft system with basic aerodynamic assumptions and a thrust model. Numerical integration is performed with a fixed-step RK4 integrator. Plots show how wind conditions (headwind/tailwind/no wind) affect fuel flow across altitudes.

Source & how to run

See the repo for the Python scripts. Run: python3 simulate.py (requirements: numpy, matplotlib). Output images are generated to the plots/ directory.
