# Saimn Zuhairi — Portfolio (Aerospace)

Live site: https://saimn05.github.io

## Structure
- Semantic HTML + Bootstrap 5 grid/utilities + small custom CSS
- One deferred JS (`script.js`) handling starfield, transitions, lightbox, GitHub repos
- Pages: Home, Projects, Experience, Contact (+ résumé PDF)

## Rubric alignment (HD 85–100%)
- Bootstrap grid/utilities used for layout (containers/rows/cols and spacing utilities)
- Accessible: semantic landmarks, skip link, focus outlines, alt text, section headings
- Performance: lazy images + width/height (reduced CLS), deferred JS, no heavy libs
- Requirements: multipage, responsive, non‑pixelated media, résumé download, tested, validated, hosted on GitHub Pages

## Files of interest
- Conceptual_Understanding.md — 300–500 word rationale
- UX_Testing_Summary.md — tasks, findings, implemented changes
- Validation_and_QA.md — validators used and results
- resume.pdf — downloadable résumé

## Local run
Open `index.html` in a browser. No build needed.

## Notes
If you test with W3C, the lightbox image uses a data‑URI placeholder `src` to satisfy the validator.
